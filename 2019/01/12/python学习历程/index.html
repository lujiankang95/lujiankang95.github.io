2、集合 set

1、无序
    1） {1,2,3}
2、不重复
3、常见操作
    1）len({1,2,3,4})
    2) 1 in {1,2,3,4}

3、字典 dict

1、由多个{key1:value1,key2:value2}组成
2、无序（不是序列）
3、根据key获取value {key1:value1,key2:value2}[key1]
4、字典中不允许有相同的键存在，键相同，值覆盖.
5、value的数据类型可以是但不限于：str int float list set dict
6、key的数据类型必须是不可变的数据类型。 int str tuple
7、定义一个空的字典 {}
